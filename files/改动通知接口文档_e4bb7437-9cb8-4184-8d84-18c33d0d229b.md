## 商品改动通知接口

> 请求地址: /api/v0.1/change_notice

> 请求方式: POST

| 参数名称            | 数据类型 | 参数描述            | 是否必须 | 示例                                   |
| ------------------- | -------- | ------------------- | -------- | -------------------------------------- |
| change_type         | int      | 改动类型            | 是       | 1.库存修改   2.价格修改   3.上下架修改 |
| change_time         | string   | 修改时间            | 是       | YYYY-mm-dd HH-MM-SS                    |
| secretId            | string   | 由被通知人提供,密钥 | 是       | AKIDz8krbsJ5yKBZQpn74WFkmLPx3gnPhESA   |
| product_id          | int      | 被修改商品id        | 是       |                                        |
| product_name        | string   | 被修改商品名        | 是       |                                        |
| brand_id            | int      | 所属品牌id          | 是       |                                        |
| brand_name          | string   | 所属品牌名称        | 是       |                                        |
| modified_field_name | string   | 被修改的字段名      | 是       |                                        |
| value_before_modify | string   | 改动前的值          | 是       |                                        |
| value_after_modify  | string   | 改动后的值          | 是       |                                        |
